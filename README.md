# AI-Exercise-Muscle-Finder
AI Exercise Muscle Finder is a web-based fitness assistant built using HTML, CSS, and JavaScript. The application helps users understand which muscle groups are targeted by specific exercises. When a user searches or speaks the name of an exercise.
AI Exercise Muscle Finder is a simple web application that helps users understand which muscles are targeted by different exercises. The app highlights the affected muscle on a human body diagram and displays an animated exercise demonstration.

## Features

- Human body diagram with highlighted muscles
- Exercise search functionality
- Voice input for searching exercises
- Exercise demonstration GIFs
- Simple and interactive user interface
- Beginner-friendly fitness visualization

## Technologies Used

- HTML5
- CSS3
- JavaScript
- SVG Graphics
- Web Speech API (Voice Recognition)

## How It Works

1. The user types or speaks the name of an exercise.
2. The system searches for the exercise in the internal database.
3. The targeted muscle group is highlighted on the body diagram.
4. An animated GIF demonstrates the exercise movement.
